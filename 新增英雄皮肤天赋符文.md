- `path` 模块未找到  
- `child_process` 模块未找到

- `node_modules/electron/index.js`
- `src/plugins/translate.js`
- `src/store/actions/systemActions.js`
- `src/store/index.js`
- `src/store/utils/socketManager.js`

## 📝 修复建议

### 方案1：修改 vue.config.js 添加 polyfills
```javascript
const path = require('path');

module.exports = {
  configureWebpack: {
    resolve: {
      fallback: {
        fs: false,
        path: require.resolve('path-browserify'),
        child_process: false
      }
    }
  }
}
```

```bash
npm install path-browserify
```

