## 四、修复说明

### 已修复的问题

#### 1. 引燃 (Ignite)
- **问题**: 缺少治疗降低效果，伤害数值不正确
- **修复**: 
  - 增加`StatsModifier.HealAndShieldBonus.PercentBonus = -0.5f`实现50%治疗降低
  - 伤害数值从`10 + Level * 4`修正为`20 + Level * 10`

#### 2. 虚弱 (Exhaust)
- **问题**: 缺少伤害降低效果，数值不正确
- **修复**:
  - 移速降低从30%修正为40%
  - 攻速降低从30%修正为40%
  - 新增`SpellDamageBonus`和`AttackDamage`降低70%效果
  - 移除不正确的护甲/魔抗降低效果

#### 3. 治疗 (Heal)
- **问题**: 治疗数值和等级计算方式不正确
- **修复**:
  - 治疗数值从`75 + Level * 15`修正为`70 + Level * 10`
  - 施法者等级计算修正（使用owner等级而非target等级）
